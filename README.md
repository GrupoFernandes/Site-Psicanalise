# Site Psicanálise — Iolanda Fernandes

Site profissional de psicanálise para Iolanda Fernandes.

## 📋 Status do projeto

🚧 **Em planejamento** — aguardando informações da profissional para iniciar o desenvolvimento.

## 🎯 Objetivo

Site institucional / portfólio profissional (página de apresentação), **sem sistema dinâmico** (sem login, sem banco de dados, sem cadastros). Apenas conteúdo visual e informativo.

## 🛠 Stack técnica

- **HTML5** + **CSS3** + **JavaScript** puro (site estático)
- Sem frameworks pesados — site rápido, leve, mobile-first
- Compatível com **GitHub Pages** (hospedagem gratuita)

## 🌐 Hospedagem

- **Plataforma**: GitHub Pages (gratuita)
- **Repositório**: https://github.com/GrupoFernandes/Site-Psicanalise
- **Domínio**: a ser definido (provavelmente `.com.br` adquirido no Registro.br)
- **Custo total**: apenas o domínio (~R$ 40-50/ano), zero mensalidade

## 📐 Seções planejadas (one-page)

1. **Hero** — foto + nome + frase acolhedora + botão "Agendar conversa inicial"
2. **Sobre** — bio profissional + formação
3. **Atendimento** — modalidades (online/presencial), público, abordagem
4. **Áreas de atuação** — cards com os temas (ansiedade, luto, autoconhecimento, etc)
5. **Como funciona** — primeira sessão, frequência, sigilo
6. **Contato** — WhatsApp + e-mail + Instagram + endereço

## 🎨 Diretrizes visuais (provisórias)

- **Paleta**: tons sóbrios e acolhedores (verde-sálvia, azul-acinzentado, terracota suave, off-white)
- **Tipografia**: serifa elegante (Playfair Display ou Lora) + sans-serif limpa (Inter ou Lato)
- **Imagens**: foto profissional + ilustrações ou fotos abstratas/poéticas
- **Layout**: espaços generosos, calma visual, mobile-first

## ⚠️ Considerações éticas / legais

Verificar se a Iolanda é **psicanalista apenas** ou também **psicóloga formada com CRP**:

- Se tiver **CRP**: seguir **Resolução CFP 06/2019** (não prometer resultados, não exibir depoimentos de pacientes, sem termos comerciais agressivos)
- Se for **só psicanalista**: publicidade mais livre, mas manter sobriedade (psicanálise é área que valoriza discrição)

## 🔄 Workflow de desenvolvimento

1. **Coletar informações** da Iolanda (lista acima)
2. **Desenvolver** localmente em `~/Documentos/site-iolanda/`
3. **Commits** atomicos com mensagens claras
4. **Push** para `main` no repositório GrupoFernandes/Site-Psicanalise
5. **Ativar** GitHub Pages nas configurações do repositório
6. **Apontar** domínio comprado para o GitHub Pages
7. **Validar** site em produção
